Instructions for how to run

- CA1-Writer 폴더는 과제2의 2번문제, 즉 입력받은 값을 .last파일에 갱신시키는 역할을 하는 컨테이너 입니다.

- CA1-monitor 폴더는 과제2의 1번문제, 즉 .last파일의 변화를 감지해서 .log파일에 누적/기록 하는 컨테이너 입니다.

- 두 파일 모두 build를 마치고 같은 볼륨에 mount하여 실행합니다. 순서는 무관합니다.

- string을 전달하는 방식은 기존 방식과 동일합니다. IP address가 같을 지는 모르겠지만, "curl http://172.17.0.2:8080/:string"으로 입력하면 됩니다.

- ip는 환경에 따라 달라질 수 있으니 구동되지 않으면 별도로 확인하여 입력 부탁드립니다.

- .last파일과 .log파일의 위치는 동일합니다. data폴더 안에 있습니다.

이상입니다.

한 학기동안 고생하셨습니다. 감사합니다.
